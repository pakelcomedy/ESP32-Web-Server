About This Project

This web interface is hosted directly by an ESP32 microcontroller running a Rust-based HTTP server. Leverage modern web technologies to monitor and control your embedded devices without external dependencies.

  • No external server needed
  • Ultra-low power consumption
  • Secure over HTTPS (future support)
  • Easy to extend and customize
